# Customer Churn Analysis and Data Preparation

## Overview

This project analyzes customer subscription and listening behavior data from a music streaming platform to identify patterns related to customer churn.

The workflow covers the complete data analytics process, including data collection, cleaning, transformation, exploratory data analysis (EDA), feature engineering, and preparation of datasets for predictive modeling.

## Business Objective

The goal is to understand customer behavior and prepare a high-quality dataset that can be used to predict which customers are most likely to cancel their subscription.

By identifying churn indicators early, businesses can develop retention strategies and improve customer lifetime value.

---

## Dataset

The analysis combines multiple data sources:

* Customer information
* Listening history
* Audio content metadata
* User session data

The data contains subscription details, customer activity, listening behavior, and account history.

---

## Project Workflow

### 1. Data Collection

* Imported data from CSV and Excel sources
* Combined information from multiple datasets
* Validated data structure and consistency

### 2. Data Cleaning

Performed extensive data quality checks, including:

* Data type conversion
* Missing value treatment
* Text standardization
* Duplicate removal
* Outlier identification and handling

### 3. Feature Engineering

Created new analytical variables to improve business understanding and future modeling performance, including behavioral and subscription-related metrics.

### 4. Exploratory Data Analysis (EDA)

Conducted exploratory analysis to:

* Understand customer behavior patterns
* Identify churn-related trends
* Analyze listening activity
* Evaluate customer engagement
* Detect anomalies and correlations

### 5. Data Preparation for Machine Learning

Prepared the cleaned dataset for predictive modeling by:

* Transforming variables
* Creating model-ready features
* Validating data quality
* Organizing datasets for supervised learning
